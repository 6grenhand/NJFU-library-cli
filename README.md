NJFU-library-cli 是使用rust编写的实现图书馆查询,预约,签到,签退的命令行工具

## how to use

```bash
njfulib help
```

## build from source

```bash
git clone https://github.com/jyf-111/NJFU-library-cli.git
cd NJFU-library-cli
cargo build
cargo run help
```
